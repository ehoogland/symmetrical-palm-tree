Arrow Function Writing Drills

In this exercise, you will convert function declarations and expressions to arrow functions.

Drills

  • Change the function declarations and expressions below to arrow functions.
  • Ensure that the console logs still output the same results as before the conversion.
Drills: Change the below function declarations to arrow functions

Code Challenge

  • In the JS section, convert each function declaration or function expression given to an arrow function. The resulting console.logs should be the same as before you did the conversion.

function ftToInches(ft) {
  return ft * 12;
};

const ftToInches = (ft) => ft * 12;

// Test 1
console.log(`5 feet equals: ${ftToInches(5)} inches.`);


function ftToInches(ft) {
  return ft * 12;
};
const ftToInches = (ft) => ft * 12;
// Test 2
console.log(`5 feet equals: ${ftToInches(5)} inches.`);

function area(width, length) {
  return width * length;
}
// Test 3

console.log(`The area of a 4x5 rectangle is: ${area(4, 5)}.`);

const daysToHours = function(days) {
  return days * 24;
};
const daysToHours = (days) => days * 24;

// Test 4
console.log(`28 days is equal to ${daysToHours(28)} hours.`);


function cardName(value, suit) {
  return `${value} of ${suit}`;
}

const cardName = (value, suit) => `${value} of ${suit}`;
};
// Test 5
console.log(`You turn the card over to reveal a ${cardName(7, "Clubs")}!`);
