What is a Promise?

    
Promises are a way to handle asynchronous operations in JavaScript. 
        A Promise is an object that represents the eventual completion (or failure) 
        of an asynchronous operation and its resulting value.

    
    const myPromise = new Promise((resolve, reject) => {
        let condition = true; // This is a placeholder for the condition to test
        if (condition) {
            const message = "Operation successful!"; // Log the success message
            console.log(message); // Resolve the promise with a value
            resolve(message);
        } else {
            const message = "Operation failed."; 
            console.log(message); // Log the failure message
            reject(message); // Reject the promise with a reason
        }
    });
    
    
Understanding Promises in JavaScript

    
States of a Promise

    
    
        
  • Pending: The initial state, neither fulfilled nor rejected.
    • 
        
  • Fulfilled: The operation completed successfully.
    • 
        
  • Rejected: The operation failed.
